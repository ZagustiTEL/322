# 🔐 Генератор паролей с системой авторизации

Веб-приложение на Flask с системой регистрации/авторизации и генератором безопасных паролей.

## 🚀 Функциональность

### 🔑 Система авторизации
- Регистрация новых пользователей
- Вход в систему
- Защита сессий
- Хеширование паролей (SHA-256)
- Хранение данных в текстовом файле

### 🛡️ Генератор паролей
- Настройка длины пароля (4-50 символов)
- Выбор типов символов:
  - Заглавные буквы (A-Z)
  - Строчные буквы (a-z)
  - Цифры (0-9)
  - Специальные символы (!@#$% и др.)
- Проверка сложности пароля
- Оценка безопасности
- Копирование в буфер обмена

## 🛠️ Технологии

- **Backend**: Python, Flask
- **Frontend**: HTML, CSS, JavaScript
- **Безопасность**: SHA-256 хеширование
- **Хранение данных**: Текстовый файл (users.txt)
